Capabilities

Built to actually ship code

Not a chatbot. A full agent loop. Simple Q&A routes direct — complex tasks go through plan, execute, verify, synthesize.

01 · Agent
LangGraph Workflow

ReAct loop with planning, tool execution, verification and synthesis. Short-circuits simple queries to save tokens.

02 · Vector
Codebase Indexing

Ingests your entire repo into ChromaDB. Semantic retrieval before any edit — Codi knows your architecture, not just filenames.

03 · Refine
Cognitive Refiner

Intercepts vague prompts and rewrites them using a dedicated fast model. No placeholders, no corner-cutting.

04 · Plug
MCP Plugin System

21 pre-configured MCP servers. Enable git, browser automation, databases, GitHub instantly — no code changes needed.

05 · Route
Multi-Provider LLM

Switch between Ollama, Groq, OpenAI, Anthropic, or Gemini in one line. Separate refiner and coder model roles.

06 · Safe
Sandboxed Execution

Directory traversal blocked. Dangerous patterns intercepted. Python syntax validated before any file write.

Native Tools

What Codi can do

Five native tools loaded by default. MCP extends this to 30+ depending on your config.

read_file()Read any file's full contents by path
write_file()Write content and auto-create directories
run_command()Execute shell commands with sandbox guard
search_codebase()Semantic search over ChromaDB indexed code
list_files()Recursively list files, skipping junk dirs
/index <path>Index a repository into chroma_db for RAG
Commands
/index <path>Index a directory for semantic search
/mcpList MCP servers and enabled state
/mcp on <n>Enable a server at runtime
/mcp off <n>Disable a server at runtime
/logsOpen live telemetry dashboard
/clearClear session memory
/historyShow current memory state
/quitExit Codi
Setup

Running in 5 minutes

Windows, macOS, Linux. Local mode needs Ollama. Cloud mode needs one API key.

01 — Clone
git clone https://github.com/AbheyTiwari/CODI
cd CODI/coding-agent
02 — Environment
# Windows
python -m venv .venv
.\.venv\Scripts\Activate.ps1

# macOS / Linux
source .venv/bin/activate
03 — Install
pip install -r requirements.txt
04 — Run
# add keys to .env
python main.py
