How it works

The full agent loop

Not a one-shot prompt. CODI plans, executes tools, verifies results, and synthesizes a response. Every time.

01 · Plan
Understands your task

Refines your prompt, indexes your codebase, and creates a step-by-step execution plan before touching any file.

02 · Execute
Calls real tools

Reads files, writes code, runs shell commands. Actual tool calls — not hallucinated output you have to copy manually.

03 · Verify
Checks its own work

After each execution, CODI checks whether the task was actually completed. If not, it retries with context from the failure.

04 · RAG
Knows your codebase

ChromaDB vector search over your entire project. CODI finds relevant code before editing — no hallucinated imports.

05 · Plugins
21+ MCP servers

Git, browser automation, PostgreSQL, Redis, Docker, GitHub — toggle any at runtime. No code changes needed.

06 · Providers
Any LLM, one line

Ollama locally, Groq for speed, Anthropic for quality, Gemini as fallback. Switch with MODE in config.py.

Native Tools

What CODI can actually do

5 native tools. MCP extends this to 30+ depending on your config.

read_file()Reads any file's full contents by path
write_file()Writes content, auto-creates directories
run_command()Executes shell commands with sandbox guard
search_codebase()Semantic search over ChromaDB indexed code
list_files()Recursively lists files, skipping junk dirs
/index <path>Indexes a repo into chroma_db for RAG
Commands
/index <path>Index a directory for semantic search
/mcpList MCP servers and enabled state
/mcp on <n>Enable a server at runtime
/mcp off <n>Disable a server at runtime
/logsOpen live telemetry dashboard
/clearClear session memory
/historyShow current memory state
/quitExit CODI

Setup

Running in 5 minutes

Windows, macOS, Linux. Local needs Ollama. Cloud needs one free API key.

01 — Clone
git clone https://github.com/AbheyTiwari/CODI
cd CODI/coding-agent
02 — Environment
# Windows
python -m venv .venv
.\.venv\Scripts\Activate.ps1

# macOS / Linux
source .venv/bin/activate
03 — Install
pip install -r requirements.txt
04 — Configure & Run
# add your API key to .env
# CODI_GROQ_API_KEY=your_key
python main.py
